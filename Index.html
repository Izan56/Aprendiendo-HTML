Estilos CSS en HTML

¡¡ATENCIÓN!! Estos códigos es mejor ponerlos todos en head o cuando

Utilizando (style) podemos hacer que los carácteres tengan una visualización determinada, ¡¡OJO!!, utilizar [<]style[>] y para cerrar [<]/style[>].
Un ejemplo de diferente visualización es por ejemplo cambiar el color de sólo los h2
        <style>

    h2 {
      color: orange;
    }
        </style>
Después de utilizar este código en el creador de texto/código, podrás notar un cambio en los títulos h2, en esta misma página esta configurado de esta manera.
De otra manera, si quermos hacerlo con CSS deberemos hacer un nuevo archivo ______.css y añadir en este archivo un nuevo código, por ejemplo:
      h1 {
      color: navy;
      }
De esta manera sólo los títulos h1 se cambiaran de color. Perooo, al intentar recargar el HTML verás que no cambia nada de nada, eso es normal, lo que debes hacer es poner el siguiente código para llamar al cambio de color, en este caso el archivo ______.css que tengo en mi creador se llama estilos.css que es donde tengo el anterior código, por lo tanto lo añado entre las "" y así el HTML sabrá que ese archivo tambien existe. 
          <link rel="stylesheet" href="estilos.css">
Si queremos cambiar el tipo de letra de la página HTML lo que debereremos hacer es lo siguiente:
Escribir el siguiente código en el archivo _______.css que hemos creado anteriormente para cambiar el color del título h1, que lo que hará es cambiar el estilo de letra de toda la página.
      body {
      font-family: sans-serif;
      }

Scripts en HTML

En este apartado vamos a aprender a crear scripts en un nuevo archivo Java Script, que llevará el nombre de ______.js. Una vez creado este archivo, deberás añadir el código siguiente en ese archivo: 
      console.log("¡¡Buenas!! aprendiendo a invocar código Java Script desde HTML")
Y en el archivo HTML deberemos añadir el código siguiente:
        console.log("¡¡Buenas!! estamos aprendiendo HTML")
De esta manera, al ir a nuestra página web y buscar en inspeccionar (dando click derecho) veremos todo lo que esta añadido en la página web, en ese "menú" si vamos a la sección de consola veremos los scrips que hemos añadido si todo a salido bien
